GUI приложение разработано в Qt Creator. Код написан в среде разработки Qt на языке С++17. 
Папка release содержит Person.exe, необходимые библиотечные файлы Qt и примером готового ini файла person_storage.ini. Папка src содержит исходный код программы. Программа  
позволяет осуществлять демонстрацию содержимого person_storage.ini и запись данных в него.
